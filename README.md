# Project-2-Linux-Web-Database-Server-Deployment-with-Public-Access
Project Overview :- This project demonstrates deployment of a Linux based Web and Database Server infrastructure inside a virtualized environment using VMware ESXi.

- The system includes:

     RHEL 9 server with Cockpit for remote server management

     Rocky Linux server configured as Web & Database Server

     MariaDB database

     Apache Web Server

     phpMyAdmin for database management

     Firewall and NAT configuration in pfSense for public access

-This setup simulates a real-world hosting environment where services can be securely accessed from the internet.
